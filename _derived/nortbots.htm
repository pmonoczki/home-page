FrontPage futtatáshoz szükséges összetevő lapja

Olyan űrlapot adott le, vagy olyan lapra ugrott, amelynek megfelelő működéséhez webkiszolgáló és a FrontPage Server Extensions szükséges.

Ezen űrlap vagy más FrontPage összetevő helyes működéséhez ezt a webhelyet olyan webkiszolgálón kell közzétennie, amelyen a FrontPage Server Extensions telepítve van.

Az előző lapra való visszatéréshez kattintson a <Vissza> nyílra.
